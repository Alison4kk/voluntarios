# Voluntarios


## Requisitos
- Composer
- PHP
- Node
- concurrently (Opcional)

## Importar Projeto
```bash
git clone URL_PROJECT
cd project
composer install
npm install
```

## Iniciar projeto
- Terminal 1
```bash
npm run dev
```
- Terminal 2
```bash
php artisan serve
```


##  (Opcional) Usar "concurrently"
```bash
npm install -g concurrently
```
Para Iniciar
```bash
npm start
```
